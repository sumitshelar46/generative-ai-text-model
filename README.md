# Generative AI Text Model

This project demonstrates a simple Generative AI model that produces human-like text using a transformer-based language model.

## Project Overview
The model takes a user prompt as input and generates relevant text output using a pre-trained GPT-style transformer.

## Technologies Used
- Python
- Hugging Face Transformers
- GPT-2 language model

## How It Works
1. User enters a text prompt
2. The model processes the prompt
3. AI generates a continuation of the text

## Use Cases
- Content generation
- AI chat systems
- Educational demonstrations
